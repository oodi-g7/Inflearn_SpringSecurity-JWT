## 15-2. RSA이란
### 15-2-1. RSA 암호화
- 공개키 암호시스템의 하나
- 암호화뿐만 아니라 전자서명(인증)이 가능한 최초의 알고리즘
- 두 개의 키를 사용 
    1. public key : 공개키
    2. private key : 개인키
    - 공개키로 암호화(encrypt)된 데이터 → 개인키로 복호화(decrypt) 가능
    - 반대로, 개인키로 암호화(encrypt)된 데이터 → 공개키로 복호화(decrypt) 가능
